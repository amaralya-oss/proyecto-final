# Proyecto Final - Barra de Navegación

Este proyecto es un ejercicio práctico de Git y GitHub donde se simula un flujo de trabajo colaborativo.

## Funcionalidades implementadas
- Creación de un repositorio local y remoto.
- Configuración de Git con nombre y correo.
- Creación de ramas para nuevas funcionalidades.
- Implementación de una barra de navegación básica en `index.html`.
- Merge de la rama `feature/navbar` a la rama principal `main`.

## Ramas utilizadas
- `main`: rama principal donde se fusionaron los cambios.
- `feature/navbar`: rama donde se desarrolló la barra de navegación.
